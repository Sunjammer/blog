Interactive exhibit

Statnett Nydalen, Oslo.

This installation used a pair of standard web cameras to detect motion, driving a particle system which in turn drove a custom hardware-accelerated renderer. To overcome situational hardware limitations, the renderer and physics simulation was required to run on a pair of networked clients drawing to four screens each, synchronized using deterministic techniques and a UDP clock server. The solution was written in Haxe 3 targeting Adobe AIR with Stage3D.

"Inky & Smudge"

Game engine, content creation tooling & support.
To build this children's storybook/minigame collection, I provided custom game engine and content creation tooling using Haxe 3 and the OpenFL framework. This allowed us to create and test content in a native desktop environment, as well as run development builds on convenient hardware, such as cheaper Android tablets. The engine supported bone animation, sprite sheet animation, music and audio effect playback, a high performance 2D rendering solution and rich scripting. The flexibility of the engine and tooling is evident in how it was originally designed to support bullet-hell shooting games.

Content and gameplay was implemented by John Davies.
